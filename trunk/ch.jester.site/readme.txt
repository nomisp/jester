- Deutsch -
[Installation]
Keine separate Installation erforderlich. Entzippten Ordner ins Zielverzeichnis schieben, und jester starten.

[Quick Guide]
- Inline Hilfe ist mit Taste F1 erreichbar
- Für den Spielerimport ist eine aktive Internetverbindung nötig. Bei einer inaktiven Verbindung können einzelne Funktionen deaktiviert sein.


- English -
[Installation]
An installation is not required. Just drop the unzipped folder into the target directory and launch jester.

[Quick Guide]
- Inline Help: available with key F1
- Some functions require an active internet connection. In case of an inactive connection some functions may be disabled.
